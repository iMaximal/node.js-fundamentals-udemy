# Mini notes app

## Install

````bash
npm install
````

## Usage

Add Note:
````bash
node index.js add --title="to buy" --body="food"
````

Delete Note:
````bash
node index.js remove --title="to buy"
````

Get Note:
````bash
node index.js read --title="to buy"
````

Get All Notes:
````bash
node index.js list
````
